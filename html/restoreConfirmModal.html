Confirmar Restauración

Advertencia Importante

Está a punto de restaurar la base de datos desde un archivo de respaldo. Esta operación sobrescribirá todos los datos actuales.

Al confirmar la restauración:

  • Se creará automáticamente un respaldo de seguridad de los datos actuales
  • Los datos del archivo de respaldo reemplazarán completamente la base de datos
  • Esta operación puede tardar varios minutos dependiendo del tamaño del respaldo
  • Se recomienda hacer esta operación durante horas de bajo uso del sistema
